2. Подключаем образ Windows server 2012 R2 на диск D. Устанавливаем Net framework 3.5 (не забываем указать альтернативный путь с диска). 3. Устанавливаем SQLServer 2014 SP1 (движок и средства управления). Задаем степень параллелизма для сервера равной единице. 4. Запускаем скрипт из powershell. 5. Устанавливаем все скачанные необходимые компоненты, кроме appFabric и кумулятивного обновления для нее (3-я и 4-я ссылки). 6. Подключаем к диску D образ установочного диска с Sharepoint 2016. Вводим в powershell команду: "D:\prerequisiteinstaller.exe /appFabric:C:\Users\Administrator\Downloads\WindowsServerAppFabricSetup_x64.exe", где путь справа - путь к скачанному файлу с appFabric. 7. После того, как система перезагрузится, выдаст сообщение о неустановленном кумулятивном пакете устанавливаем накопительный пакет обновления (ссылка 4). Перезагружаемся. 8. Система готова к установке sharepoint.
